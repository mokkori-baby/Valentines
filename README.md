# 💖 For My Favourite Human JG

A custom, interactive web experience built to celebrate you. This project features dynamic interactions, music, and custom animations to show exactly how much you mean to me.

---

## ✨ Features

*   **Interactive Prompts:** Dynamic elements designed to capture your reactions.
*   **Curated Reactions:** Packed with a collection of custom GIFs (featuring classic cat memes, cute animations, and playful responses).
*   **Immersive Audio:** Features background track elements (`TOOTIMETOOTIMETOOTIME.mp3`) to set the perfect mood.
*   **Clean Design:** Smooth styling and responsive layouts across mobile and desktop.

---

## 📂 Tech Stack & Structure

As seen in the project structure, this is a lightweight, frontend-driven interactive build:

*   `index.html` - The core structure and layout of the experience.
*   `style.css` - Custom design properties, animations, and typography layouts.
*   `script.js` - Interactive click handling, logical states, and event triggers.
*   `*.gif` / `*.webp` - Contextual visual assets handling dynamic feedback states.
